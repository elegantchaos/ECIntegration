Inherits from NSObject
Declared in ECPWPane.h

Overview

Preference panes are the panels that are shown one at a time in the window managed by the ECPWController class.

Each pane has an identifier, used to keep track of it internally, and a name, used for display purposes.

Panes also have an icon and a tooltip.

These values can be set from an info dictionary passed in to the pane. If keys are missing, default values are chosen, based on the name of the pane class.
